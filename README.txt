############ REQUISITOS ############


>>> Python ver. 3.12.3 (ARQUIVO NA PASTA INSTALL).

>>> Navegador Google Chrome ver. 125.0.6422.142 (ARQUIVO NA PASTA INSTALL).

>>> Selenium (Após instalação do Python e reiniciar o PC, abrir o CMD e digitar "pip install selenium e aguarda o fim da instalação").

>>> Requests (Após instalação do Python e reiniciar o PC, abrir o CMD e digitar "pip install requests e aguarda o fim da instalação").

>>> Driver Chrome (Arquivo dentro da pasta do bot, não precisa instalar, apenas deixe-o na pasta).



########### INSTRUÇÕES #############

Após a instalação dos arquivos e dependencias acima, siga os passos abaixo:

1º Abrir o arquivo main.py em um editor de codigo de sua preferencia.

2º Editar o arquivo main.py conforme as instruções no proprio arquivo e salvar.

3º No prompt CMD, acesse o caminho C:\bot-blaze\, feito isso digite o comando py main.py

4º Agora é só ir dormir e amanhã ja seras um milhonario :)
